Algoritmo del Ranking

El ranking se calcula sumando el total de puntos acumulados por cada torneo asistido. El criterio de desempate secundario es el **Puntaje de Eficiencia**, calculado como:

(Puntos Obtenidos / Puntos Posibles) * 100

Los puntos posibles por torneo se calculan de acuerdo a la dificultad teórica basada en el número de participantes (entrants) y seeding.
